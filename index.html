HTML

  • Hypertext Markup Language
  • An element is a type of content on a webpage
  • Elements are also used to divide the content on the page and to describe the content to browsers and other developers
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Declarations are not case sensitive
  • An HTML attribute defines additional information about an element
  • HTML attributes are always defined in the begining element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS Box Model - Every HTML element is essentially a box - Margin, Border, Padding, Content
  • An Inline CSS is used when a developer wants to style an element directly in the HTML file.
  • Internal Style Sheet contains rules for the webpage in the Head section of the HTML file
  • External Style Sheet is the most commonly used. It allows developers to keep all their CSS rules in a seperate file, making changes easier
  • External Style Sheets are an example of the "Seperation of Concerns" - the practice of seperating code into different files based on use

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Every git repository starts with a default Branch called "Main" which holds the final product
  • Multiple branches can exist at a single time, serving different purposes
  • A Feature Branch might hold code that is being developed for a single feature of the application. A working branch
  • A Develop Branch might hold code for multiple feature branches that haven't been incorporated into the main branch yet
  • Branches don't affect the Main branch until Approved, Tested, and Merged
  • A Pull Request (PR) is used to merge a comparison branch into Main

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A value can either be Truthy or Falsy
  • An Array is a single variable used to group related data, typically data that is related in the same way
  • Arrays are zero-indexed and sequential. The first item is always 0
  • A For loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
  • A Function is simply a set of instructions that tells the computer how to perform a certain task
